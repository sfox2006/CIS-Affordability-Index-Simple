Affordability Index

Have things gotten more or less affordable?

Pick a good, choose a time period, and this tracker shows whether its price went up faster or slower than wages. If the price went up faster than wages, that item became harder to afford. If it went up slower, it became easier to afford.

A basket lets you combine multiple goods into a single composite series. Assign a weight to each good to reflect its relative importance — for example, give housing a higher weight than accessories if you want the basket to reflect typical household spending. Weights are automatically normalised to 100%, so only the ratios between them matter, not the actual numbers you enter.

How this tracker works

This page is built for a simple question: did the price of something go up faster or slower than wages?

  • We use quarterly ABS price index data for the selected good or goods group.
  • We compare that with the Wage Price Index over the same period.
  • Both lines on the chart are reset to the same starting point so you can compare the change clearly.
  • If the item rose faster than wages, it became harder to afford. If it rose slower than wages, it became easier to afford.
  • Wage data only starts in December 2010, so earlier wage comparisons are not available.

We also keep an inflation comparison in the background, but the main focus on this page is affordability relative to wages.
